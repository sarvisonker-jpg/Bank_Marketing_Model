# Bank Marketing Data Analysis & Classification

## Project Overview
This repository contains a Python-based analysis of a bank marketing campaign dataset. The primary goal is to perform Exploratory Data Analysis (EDA) and build machine learning classification models to predict customer outcomes.

##  Dataset
* **File:** `Bank_Marketing_Dataset.csv`
* **Size:** 45,211 rows and 17 columns.
* **Features:** Includes 8 numerical features and 9 categorical features.

##  Prerequisites & Libraries
The code is built using Python in a Jupyter/Colab environment. The following libraries are required:
* `pandas` and `numpy` for data manipulation.
* `matplotlib.pyplot` and `seaborn` for data visualization.
* `scikit-learn` for machine learning modeling and evaluation.

##  Workflow Highlights
1. **Data Loading & Inspection:** Loading the dataset and evaluating its structure, data types, and missing values.
2. **Data Cleaning:** Checking for and dropping duplicate entries.
3. **Exploratory Data Analysis (EDA):** 
   * Segregation of numerical and categorical variables.
   * Generating histograms for numerical data distribution.
   * Generating count plots for categorical data distribution.
4. **Machine Learning Preparation:** Importing models like Logistic Regression, Decision Tree, and Random Forest for classification tasks.
